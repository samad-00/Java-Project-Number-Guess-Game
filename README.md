# Java-Project-Number-Guess-Game
A desktop-based Number Guessing Game developed using Java and Swing as part of my summer training internship. The project includes difficulty levels, hints, real-time feedback, guess counting, timers, and best-score storage. Designed to demonstrate core Java concepts, OOP, event handling, and GUI development.

## About
This project was developed during a summer training internship to practice Java Swing and core Java concepts. It implements an interactive GUI game with real-time feedback and score tracking.

## Features
- Difficulty: Easy (1–50), Medium (1–100), Hard (1–500)
- Real-time feedback: too high / too low / correct
- Hint button (gives a small range)
- Timer and guess counter
- Best score persistence (file-based)
- Reset and play-again functionality
